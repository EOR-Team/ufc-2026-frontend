## Why

建立 UFC 2026 前端工程化脚手架，统一技术栈（Vue 3 + Vuetify 3 + Tailwind CSS 4 + Vite 7 + Vue Router 5 + Pinia 3 + @mdi/font），为后续功能开发提供统一基础设施。

## What Changes

- 初始化 Vue 3 + Vite 7 项目
- 集成 Vuetify 3 Material Design 组件库
- 集成 Tailwind CSS 4 工具类样式
- 配置 Vue Router 5 路由
- 配置 Pinia 3 状态管理
- 集成 Material Design Icons (@mdi/font)
- 配置开发/生产环境
- 配置本地开发服务器

## Capabilities

### New Capabilities

- `frontend-scaffold`: 前端工程脚手架，包含项目初始化、依赖配置、开发服务器、生产构建、部署配置

### Modified Capabilities

- (无)

## Impact

- 新建 `src/` 目录结构
- 新增依赖：`vue`, `vuetify`, `tailwindcss`, `@mdi/font`, `vue-router`, `pinia`, `vite`
- 新增开发/构建配置
